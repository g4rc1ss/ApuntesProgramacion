## Corregir la ejecución infinita

En el editor de código, se define un bucle `while` dentro de una [función](course://Functions/Definition). Por favor, no te preocupes por ello en este momento,
aprenderás sobre las funciones en la siguiente sección. La forma en que está escrito en este momento, el bucle while dentro de esta función
se ejecutaría indefinidamente
porque la condición siempre sería `True`. Puedes verlo por ti mismo ejecutando el código. Para terminar la ejecución,
utiliza el botón rojo de detener en el lado izquierdo de la ventana de ejecución.
