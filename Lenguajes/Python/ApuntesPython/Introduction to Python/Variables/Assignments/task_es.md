## Tareas

Una asignación aumentada es una única sentencia que combina una operación binaria y una 
sentencia de asignación, como `+=`, `-=`, etc.

Una expresión de asignación aumentada como `x += 1` se puede reescribir como `x = x + 1` para lograr un efecto similar.
Puedes leer más sobre esto <a href="https://docs.python.org/3/reference/simple_stmts.html#augmented-assignment-statements">aquí</a>.

### Tarea
Usa una asignación aumentada para agregar `5` a `number` y actualizar la variable.

<div class='hint'>Usa el operador <code>+=</code>.</div>
