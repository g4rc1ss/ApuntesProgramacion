=================

Para utilizar la función 'impar'
---------------------------------

La función impar(n) devuelve:

    - True si número es impar o
    - False si no lo es

Primero importamos la función:

    >>> from DocTest import impar

Ejemplos de uso:

    >>> impar(0)
    False
    >>> impar(1)
    True
    >>> impar(2)
    False
    >>> impar(3)
    True
